はじめてのレスポンシブ Web ページ

2025-12-04 / カテゴリ: Web制作メモ

このページは、HTML / CSS / JavaScript だけで構成された シンプルなブログ風レイアウトの例です。画面が狭いときは 上から 記事 → サイドバー の順に縦並び、 画面が広いときは 記事 + サイドバーの2カラム に切り替わります。

サイドバーには、プロフィールと、JavaScript の Date API を使って生成した 「現在の月のカレンダー」を表示しています。 API や日付情報を扱う練習としても使えます。

このサンプルで試せること

  • 基本的なレスポンシブレイアウト（flex + media query）
  • ブログ風の見出し・本文レイアウト
  • JavaScript で DOM を操作してカレンダーを描画

コードを少しずつ書き換えて、色・フォント・構成を 自分好みにカスタマイズしてみてください。
